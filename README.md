# 诡秘世界自动化测试工具

#### 介绍
用于诡秘世界MUD开发过程的自动化测试

#### 软件架构
软件架构说明
客户端：tintin++/mudlet
控制脚本：PYTHON3

使用tintin++/mudlet客户端对服务端MUD进行测试
使用PYTHON3对测试文本结果进行分析

ttt++.sh 作为tintin++客户端启动脚本
